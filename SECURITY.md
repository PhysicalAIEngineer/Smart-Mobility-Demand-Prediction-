# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Yes              |
| < 1.0   | ❌ No               |

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

### 📩 How to Report
- Email: chetansonigara01@gmail.com
- Or open a **private issue** (if enabled)

### 📝 What to Include
Please provide:
- Description of the vulnerability  
- Steps to reproduce  
- Potential impact  
- Suggested fix (if any)  

---

## Response Timeline

- Initial response: **within 48 hours**  
- Resolution time: Depends on severity  
- Critical issues will be prioritized  

---

## Disclosure Policy

- Please do **not publicly disclose** vulnerabilities until they are resolved  
- Once fixed, we may acknowledge your contribution  

---

## Security Best Practices

- Avoid exposing sensitive data  
- Keep dependencies updated  
- Validate all inputs  
- Use secure coding standards  

---

## Scope

This project is focused on:
- Machine Learning models  
- Data preprocessing pipelines  
- Visualization components  

Security considerations mainly include:
- Data integrity  
- Dependency vulnerabilities  
- Code safety  

---

## Acknowledgements

We appreciate responsible disclosure and contributions that improve the security of this project.
